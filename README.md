# Bank Customer Churn Analysis

## Overview
This project analyzes bank customer data to identify patterns and factors that contribute to customer churn. The goal is to understand customer behavior and highlight insights that can help improve customer retention.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Churn vs Non-Churn comparison
- Customer demographic analysis
- Visualization of churn patterns

## Key Insights
- Inactive customers are more likely to churn.
- Customers with fewer bank products show higher churn.
- Certain age groups have higher churn rates.

## Conclusion
The analysis helps identify important factors influencing customer churn and provides insights that banks can use to improve retention strategies.
